# Englishom Live Dashboard - لوحة البيانات الحية

## 🎯 نبذة عن المشروع

**Englishom Live Dashboard** هي لوحة بيانات حية احترافية لمنصة Englishom لتعليم اللغة الإنجليزية. تتميز بتصميم عصري بألوان نيون وخريطة عربية تفاعلية تعرض ومضات من 85+ مدينة عربية.

---

## ✨ المميزات الرئيسية

### 1. 🎨 تصميم احترافي بألوان نيون
- شعار Englishom بألوان نيون (أصفر وأحمر وسماوي)
- تأثيرات توهج نيون على جميع العناصر
- خلفية داكنة احترافية

### 2. 🗺️ خريطة عربية تفاعلية
- تعرض جميع الدول العربية
- 85+ مدينة عربية
- ومضات متزامنة من جميع المدن
- الرياض مميزة بلون ذهبي

### 3. 📱 منصات تواصل اجتماعي
- Facebook, Twitter, Instagram, YouTube, LinkedIn
- TikTok, Snapchat, Telegram, WhatsApp
- روابط مباشرة إلى حسابات Englishom الرسمية

### 4. 🌍 ترجمة كاملة
- واجهة عربية كاملة
- واجهة إنجليزية كاملة
- تبديل سلس بين اللغات

### 5. 📊 إحصائيات حية
- عدد المستخدمين المسجلين
- المستخدمين النشطين حالياً
- أكثر الدول استخداماً
- آخر تسجيل

---

## 🛠️ التكنولوجيا المستخدمة

- **React 19** - مكتبة واجهة المستخدم
- **TypeScript** - لغة البرمجة
- **Tailwind CSS 4** - نظام التصميم
- **Express 4** - خادم الويب
- **tRPC 11** - API type-safe
- **Vite 5** - أداة البناء
- **Vitest 2** - اختبارات الوحدة

---

## 📦 المتطلبات

- Node.js 18+
- pnpm 8+

---

## 🚀 البدء السريع

### 1. استنساخ المشروع
```bash
git clone <repository-url>
cd englishom-dashboard
```

### 2. تثبيت المتطلبات
```bash
pnpm install
```

### 3. تشغيل خادم التطوير
```bash
pnpm dev
```

الموقع سيكون متاحاً على: `http://localhost:3000`

### 4. البناء للإنتاج
```bash
pnpm build
pnpm preview
```
