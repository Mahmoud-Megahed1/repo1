# دليل المبرمج - Englishom Live Dashboard

## 📋 نظرة عامة على المشروع

هذا المشروع عبارة عن **لوحة بيانات حية احترافية** لمنصة Englishom لتعليم اللغة الإنجليزية.

### المميزات الرئيسية:
- ✅ شعار Englishom بألوان نيون (أصفر وأحمر وسماوي)
- ✅ خريطة عربية تفاعلية بـ 85+ مدينة
- ✅ ومضات متزامنة من جميع المدن (كل ثانية)
- ✅ 9 منصات تواصل اجتماعي (Facebook, Twitter, Instagram, YouTube, LinkedIn, TikTok, Snapchat, Telegram, WhatsApp)
- ✅ ترجمة كاملة (عربي/إنجليزي)
- ✅ تصميم احترافي وجذاب بألوان نيون

## 🚀 البدء السريع

### 1. التثبيت
```bash
cd englishom-dashboard
pnpm install
```

### 2. تشغيل خادم التطوير
```bash
pnpm dev
```

الموقع سيكون متاحاً على: `http://localhost:3000`

### 3. تشغيل الاختبارات
```bash
pnpm test
```

### 4. التحقق من TypeScript
```bash
pnpm tsc --noEmit
```

## 🎨 الألوان والتصميم

### نظام الألوان النيون:
- **أصفر نيون:** `#FFFF00` (الرياض والمدن الرئيسية)
- **أحمر نيون:** `#FF0000` (الشعار)
- **سماوي نيون:** `#00FFFF` (الحدود والتفاصيل)
- **أسود داكن:** `#0f172a` (الخلفية)

### الخريطة:
- خريطة عربية بحدود نيون خضراء
- 85+ مدينة عربية بومضات متزامنة
- الرياض مميزة بلون ذهبي

## 📊 مكونات رئيسية

### 1. ArabicMapImage.tsx
**الوصف:** خريطة عربية تفاعلية مع ومضات متزامنة

**المميزات:**
- 85+ مدينة عربية
- ومضات متزامنة كل ثانية
- الرياض مميزة بلون ذهبي
- 8 ألوان نيون مختلفة للمدن

**الإحداثيات:**
- تم حسابها بدقة على الخريطة الأصلية
- تدعم جميع دول الخليج والشام ومصر واليمن

### 2. EnglishomLogo.tsx
**الوصف:** شعار Englishom بألوان نيون

**المميزات:**
- شعار بألوان نيون (أصفر وأحمر وسماوي)
- تأثير توهج نيون
- حجم قابل للتخصيص

### 3. Footer.tsx
**الوصف:** أيقونات المنصات الاجتماعية

**المنصات:**
- Facebook, Twitter, Instagram, YouTube, LinkedIn
- TikTok, Snapchat, Telegram, WhatsApp

**الروابط:**
- كل أيقونة لها رابط مباشر إلى حساب Englishom الرسمي
