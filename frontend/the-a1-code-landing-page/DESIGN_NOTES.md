# Design Notes - The A1 Code Landing Page

## Inspiration from Englishom.com

### Color Palette (Extracted from Englishom.com)
- **Primary Brand Colors:**
  - Dark Maroon/Burgundy: #3D1D1C (background in logo)
  - Cream/Light Yellow: #F5BB41 (accent, prominent in UI)
  - Sky Blue: #2167D1, #3D84F3 (secondary accent)
  - Green: #4CA853, #398039 (success/positive action)

- **Supporting Colors:**
  - Dark Slate: #222222, #141414, #000000 (text)
  - Light Gray: #F1F1F1, #e8e8e8 (backgrounds)
  - Bright Green: #00CC76, #66E066 (highlights)
  - Various accent colors for diversity

### Logo Analysis
- **Style:** Geometric, modern, minimalist
- **Icon:** House/Home shape with layered triangular design
- **Typography:** Bold, sans-serif (appears to be a custom or heavy weight sans)
- **Color Scheme:** Cream (#F5BB41) on dark maroon (#3D1D1C)
- **Symbolism:** Home, foundation, learning structure

### Design Philosophy for The A1 Code
- **Tone:** Professional, educational, empowering
- **Aesthetic:** Modern minimalism with geometric elements
- **Typography:** Bold display fonts paired with readable body fonts
- **Layout:** Asymmetric, dynamic (avoid centered grids)
- **Interaction:** Smooth transitions, clear CTAs
- **Depth:** Subtle shadows and layering for dimension

### Key Design Decisions
1. **Color Scheme:** Use cream (#F5BB41) as primary accent, dark navy/maroon as background
2. **Typography:** Combine bold sans-serif for headers with readable sans-serif for body
3. **Visual Elements:** Geometric shapes, layered designs (inspired by logo)
4. **Layout:** Hero section with asymmetric layout, feature cards with depth
5. **CTAs:** Prominent buttons with hover effects
