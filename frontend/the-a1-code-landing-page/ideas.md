# تصاميم صفحة الهبوط - The A1 Code

## الفكرة المختارة: التصميم الهندسي المعاصر

### فلسفة التصميم
**الحركة الفنية:** الحداثة الهندسية المتقدمة (Contemporary Geometric Modernism)

**المبادئ الأساسية:**
1. **الطبقات الهندسية:** استخدام الأشكال الهندسية المتداخلة (مثلثات، مربعات) لإنشاء عمق بصري
2. **التسلسل البصري الواضح:** تدفق منطقي من الأعلى للأسفل مع نقاط توقف قوية
3. **الحركة والديناميكية:** تجنب التخطيطات المركزية المملة، استخدام عناصر غير متماثلة
4. **الوضوح التعليمي:** كل عنصر يخدم غرضاً تعليمياً واضحاً

### فلسفة الألوان
**المنطق العاطفي:** الألوان تعكس رحلة التعلم من الأساس إلى الإتقان
- **الكريمي الذهبي (#F5BB41):** الدفء والثقة والنجاح
- **الأزرق الداكن (#1F6BF6):** الثقة والاستقرار والاحترافية
- **الأسود الناعم (#222222):** الأناقة والتركيز
- **الأبيض النقي (#FFFFFF):** الوضوح والبساطة

### نموذج التخطيط
**البنية غير المتماثلة:**
- Hero section بزاوية ديناميكية مع نص على اليسار والشعار على اليمين
- Feature cards بتصميم متدرج وليس شبكة منتظمة
- Sections تتناوب بين الخلفيات الداكنة والفاتحة

### العناصر المميزة
1. **الخطوط الهندسية:** خطوط قطرية تفصل بين الأقسام
2. **الشعار المحسّن:** نسخة معاد تصميمها من شعار Englishom مع تأثيرات طبقية
3. **الزوايا الديناميكية:** استخدام clip-path لإنشاء حواف غير منتظمة

### فلسفة التفاعل
- **الانتقالات السلسة:** كل عنصر يتحرك بسلاسة عند التفاعل
- **تأثيرات الـ Hover:** ارتفاع العناصر وتغيير الألوان
- **الرسوم المتحركة عند الدخول:** العناصر تظهر تدريجياً عند تحميل الصفحة

### الرسوم المتحركة
- **الدخول:** Fade-in مع slide-up للعناصر الرئيسية
- **الـ Hover:** Scale up وتغيير الظل للأزرار والبطاقات
- **التمرير:** عناصر تتحرك بسرعات مختلفة (Parallax effect)

### نظام الطباعة
- **العناوين الكبيرة:** Poppins Bold (700) - جريئة وحديثة
- **العناوين الفرعية:** Poppins SemiBold (600)
- **النص الأساسي:** Segoe UI / System Font (400) - قابل للقراءة
- **الاقتباسات:** Poppins Medium (500) - مائل
